RCon клиент для сервера minecraft.

Поддерживается Bukkit, Spigot, Sponge и другие ядра, имеющие поддержку RCON.
Hubs.RCon позволяет легко получить удалённый доступ к консоли сервера.

Hubs.RCon используется через командную строку следующим образом:
java -jar <путь до Hubs.RCon> <IP сервера> <Порт для RCon> <Пароль от RCon>
